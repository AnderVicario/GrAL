## ğŸ— Instalazioa eta exekuzioa  

### ğŸ”¹ Aukera 1: Docker bidez  

1. Irudia sortu:  
   ```sh
   docker-compose build
   ```

2. Aplikazioa abiatu:  
   ```sh
   docker-compose up -d
   ```

3. Aplikazioa gelditu:  
   ```sh
   docker-compose down
   ```

### ğŸ”¹ Aukera 2: Python inguruneen bidez

1. Ingurune birtuala sortu:  
   ```sh
   python3 -m venv venv
   ```

2. Ingurunea aktibatu:  
   Windows:
   ```sh
   .\venv\Scripts\activate
   ```
   Linux/Mac:
   ```sh
   source venv/bin/activate
   ```

3. Paketeak instalatu:  
   ```sh
   pip install -r requirements.txt
   ```

4. Aplikazioa exekutatu:  
   ```sh
   python app.py
   ```
